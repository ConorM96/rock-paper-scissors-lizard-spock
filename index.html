Introduction

Rock, Paper, Scissors, Lizard, Spock
is an expansion on the original version
"Rock, Paper, Scissors", which provides
more options to choose from,
which will result in less draws between
players making the game more entertaining.

X

Rules

In Rock, Paper, Scissors, Lizard, Spock:
Rock crushes Scissors,
Scissors cuts Paper,
Paper covers Rock,
Lizard poisons Spock,
Spock smashes Scissors,
Scissors decapitates Lizard,
Lizard eats Paper,
Paper disproves Spock,
Spock vaporizes Rock,
Rock crushes Lizard.

First player to 3 points wins.
